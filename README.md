# Advanced Concept Mapping App

A powerful and intuitive web application for creating interactive concept maps and mind maps. Built with modern web technologies to help you visualize and organize complex ideas.

## âœ¨ Features

- **Interactive Canvas**: Drag-and-drop interface with zoom and pan controls
- **Custom Shapes**: Multiple node shapes (circle, rectangle, diamond, hexagon, star, triangle)
- **Flexible Connections**: Create relationships between concepts with customizable edges
- **AI Assistant**: Get suggestions and help organizing your concepts
- **Multiple Layouts**: Tree, radial, network, histogram, and free-flow layouts
- **Rich Customization**: Colors, icons, tags, and notes for each node
- **Export/Import**: Save and share your maps in JSON or CSV format
- **Undo/Redo**: Full history support for all changes
- **Responsive Design**: Works on desktop and tablet devices
- **Theme Support**: Light and dark mode
